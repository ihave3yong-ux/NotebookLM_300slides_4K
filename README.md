# 🎨 NotebookLM Ultimate Design Hub (300)

노트북LM 슬라이드 디자인 고민 탈출! 
웹앱(Web App) 하나로 슬라이드 디자인의 모든 것을 해결하는 끝판왕 애플리케이션(Application)입니다.

## ✨ 가장 강력한 핵심 기능 (Core Features)
- **🌟 총 300종의 압도적인 디자인 템플릿(Template)**: 20가지 카테고리(Category)별로 각각 15종류의 완벽한 4K 해상도(Resolution) 슬라이드 디자인 프롬프트(Prompt)가 기본 제공됩니다.
- **🤖 무한정 개인별 AI 맞춤형 디자인(Custom Design)**: 300종의 기본 템플릿에 더하여, 사용자 개인의 요구에 완벽하게 맞춘 '무한정 AI 맞춤형 디자인 서비스'가 가능합니다!

## 🚀 추가 주요 기능 (Additional Features)
- **원클릭 타겟팅 시스템(One-Click Audience Targeting System)**: 단 한 번의 클릭으로 청중(Audience)에 맞는 디자인 타겟팅(Targeting) 적용
- **직관적인 사용자 환경(Intuitive UI/UX)**: 누구나 쉽게 원하는 디자인 프롬프트를 찾고 복사할 수 있는 최적의 환경 제공

## 🛠 사용된 기술 (Tech Stack)
- **프론트엔드(Frontend)**: React.js
- **아이콘(Icons)**: Lucide-react
- **스타일링(Styling)**: Modern CSS (반응형 웹(Responsive Web))
